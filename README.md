# 🔌 12V DC Power Supply Project

A simple and reliable **12V DC regulated power supply** built using basic electronic components. This project converts AC mains voltage into a stable **12V DC output** suitable for powering small electronics and circuits.

---

## 📌 Overview

This project demonstrates how to design and build a **linear DC power supply** using a transformer, rectifier, filter, and voltage regulator. It is ideal for beginners learning power electronics and circuit design.

---

## ⚙️ Features

- ✔️ Converts AC to stable **12V DC output**
- ✔️ Uses **7812T voltage regulator**
- ✔️ Smooth DC output using filtering capacitors
- ✔️ Simple and cost-effective design
- ✔️ Beginner-friendly project

---

## 🧰 Components Used

- Step-down Transformer (**12-0-12V**)
- Diodes (Bridge Rectifier)
- Capacitors (Filtering)
- **7812T Voltage Regulator**
- Resistor (**1kΩ**)
- Potentiometer (**10kΩ**)
- Wires
- PCB / Breadboard

---

## 🔧 Working Principle

1. **Step-down Transformer**  
   Reduces high AC voltage to low AC voltage (12V).

2. **Rectification**  
   Diodes convert AC into pulsating DC using a bridge rectifier.

3. **Filtering**  
   Capacitors smooth the output by reducing ripples.

4. **Voltage Regulation**  
   The **7812T** ensures a stable and adjustable DC output.
